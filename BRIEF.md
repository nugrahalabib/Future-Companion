## Hasil Brainstorm dengan Gemini

Konsep Utama: "The Sovereign Companion" (Tahun 2076)
Premis: Manusia sudah mencapai titik lelah batin (social fatigue) dalam berinteraksi dengan sesama manusia. Sistem ini hadir sebagai solusi absolut: Companion buatan yang 100% didesain untuk mengerti, menemani, dan memenuhi kebutuhan fisik maupun emosional user tanpa friksi.

Berikut adalah jabaran detail alur demonya:

### 1. The Hook: Immersive Landing Page
Visual & Copywriting: Layar dibuka dengan visual sinematik resolusi tinggi yang menunjukkan kontras: dunia luar yang dingin dan berjarak, versus kehangatan berada di rumah bersama companion yang menatap penuh kasih.

Value Proposition: "Why compromise when you can customize your reality?"

Call to Action (CTA): Tombol interaktif yang menyala subtle: "Initialize Your Companion".

### 2. Registration & Data Harvesting (The Anchor)
Formulir Real-Data: User diminta memasukkan data asli (Nama, Email, Usia, Pekerjaan, Status Hubungan saat ini).

Psikologi UI: Desain dibuat sangat elegan dan terkesan seperti pendaftaran eksklusif atau waitlist produk high-end, sehingga user secara psikologis rela memberikan data riil mereka. Data ini menjadi pondasi awal metrik audiens untuk riset.

### 3. The Creator Studio: Deep Customization Engine
Ini adalah core experience dari sisi visual sebelum masuk ke AI. UI harus terlihat seperti dashboard laboratorium bio-sintetis masa depan.

Anatomy & Biomechanics (Fisik):

Slider presisi untuk rasio wajah (garis rahang, jarak mata, proporsi hidung).

Bentuk tubuh, postur, warna kulit, tingkat kehangatan suhu tubuh sintetis, dan tekstur rambut.

Fitur Biologis Ekstrem: Opsi toggle switch untuk "Artificial Womb (Uterus Sintetis)" bagi robot wanita, atau "Synthetic DNA/Sperm Bank Integration" bagi robot pria. Detail ekstrem ini akan menjadi shock factor yang menunjukkan kemajuan radikal industri ini.

Persona & Psyche (Mental):

Penentuan Role: Pasangan romantis, asisten dominan, pendengar yang pasif, atau partner debat yang intelektual.

Trait Sliders: Tingkat kepatuhan (Submissive vs Dominant), selera humor (Dry/Sarcastic vs Slapstick), tingkat agresivitas (Naughty vs Formal/Innocent).

Hobi dan Knowledge Base: Database ketertarikan (Seni, Teknologi, Filosofi, Olahraga).

Data Capture Point: Setiap pergeseran slider dan toggle yang dipilih user akan langsung dicatat ke database via API sebagai metrik "Market Demand Preferences".

### 4. The Encounter: 10-Minute Live Test
Setelah selesai merakit, transisi layar masuk ke mode "Awakening". Layar menampilkan simulasi ruangan, dan robot hasil custom tersebut perlahan menatap langsung ke arah user menunggu input.

Teknologi: Di sinilah API Gemini 2.5 Flash Native Audio bekerja. Streaming audio dua arah memastikan latency nyaris nol, membuat interaksi suara terasa seperti ngobrol dengan manusia sungguhan.

Prompting Dinamis: Konfigurasi persona dari tahap 3 di-parse menjadi payload JSON dan disuntikkan langsung ke dalam system prompt Gemini. Jika user set "Centil dan Manja", intonasi suara dan gaya bicaranya langsung beradaptasi seketika.

Timer & Analytics: Ada countdown 10 menit. Selama obrolan, speech-to-text process bekerja di background untuk menyimpan transkrip. Transkrip ini otomatis dianalisis (sentiment & topic analysis) untuk memetakan arah minat obrolan user (contoh: persentase topik emosional, seksual, atau kasual).

### 5. The Checkout: 10-Second Instant Delivery
Jika user sudah merasa cukup atau waktu habis, layar berpindah ke penawaran subscription.

Copywriting: "Your companion is ready. Initiating Instant Matter-Transfer Delivery."

Payment & Address: Muncul QR Code dinamis untuk simulasi potong saldo. User mengetik titik kordinat/alamat pengiriman.

The Countdown: Setelah scan simulasi berhasil, layar menampilkan animasi futuristik pengiriman dengan countdown raksasa: 10, 9, 8... hingga 0.

The Reveal: Muncul instruksi on-screen: "Delivery Complete. Please check the package in front of you."

### 6. The Physical Bridge (Kejutan Pameran)
Langkah ini menggabungkan software dengan hardware physical experience.

Tim di booth pameran sudah menyiapkan mini-box fisik elegan yang diletakkan di dekat user tanpa mereka sadari sebelumnya.

Di dalam box tersebut terdapat barang fisik: stiker tutorial, kartu garansi biomekanik fiktif, dan manual book "How to Boot Up Your Companion". Ini memberikan efek penutup demonstrasi yang teatrikal dan sangat memorable bagi investor.

### 7. Confirmation & The Harvard-Grade Questionnaire
User kembali ke layar PC dan mengklik tombol "Package Received".

Layar bertransisi menjadi Post-Demo Assessment. Karena user sudah mendapatkan pengalaman end-to-end yang imersif, retensi untuk mengisi form akan sangat tinggi.

Pertanyaan Komprehensif: Skala kepuasan interaksi suara AI, kemudahan UI kustomisasi, feedback kelayakan konsep di dunia nyata, dan kesediaan menukar interaksi manusia dengan robot komersial. Output data kualitatif dan kuantitatif ini akan diekspor untuk material jurnal ilmiah kelas dunia.

### 8. The Investor Admin Dashboard (The Control Center)
Sistem ini dirangkum dalam backend web app khusus untuk tim project.

Live Traffic & Conversion Rates: Memantau berapa user yang register versus berapa yang menyelesaikan tahap checkout.

Preference Heatmap: Visualisasi grafik yang menunjukkan tren kustomisasi. Memudahkan tim melihat matriks seperti "80% user mengaktifkan fitur Uterus Sintetis" atau "Mayoritas memilih trait Dominan".

Conversational Insights: Word cloud dan grafik analisis topik dari obrolan API Gemini, memberikan insight mentah tanpa filter tentang apa yang manusia cari dari sebuah AI companion.

## Detailed Prompt Brief (for Claude Code execution)

### CONTEXT & ROLE
Act as an Expert Full-Stack Developer and UI/UX Visionary specializing in highly immersive, futuristic web applications. Your task is to build a fully functional, end-to-end prototype for a project called "The Sovereign Companion". This prototype will be demonstrated at the "Tech AI Future Indonesia" exhibition to secure investor funding. 

The premise is set 50 years in the future (2076) where human-to-human relationships have severely declined, and users purchase hyper-customizable AI humanoid companions.

### TECH STACK REQUIREMENTS
- Frontend: Next.js (React), Tailwind CSS, Framer Motion (for highly fluid, cinematic animations).
- Backend: Node.js/Express or Next.js API Routes.
- Database: PostgreSQL (via Prisma ORM) to capture strict relational data for analytics.
- Voice AI Integration: Gemini 2.5 Flash Native Audio Preview (Live API) `gemini-2.5-flash-native-audio-preview-12-2025` via WebRTC/WebSocket for ultra-low latency voice conversation.
- State Management: Zustand or Redux Toolkit.
- Deployment: Dockerized. Ensure the architecture is optimized to run smoothly on a local environment or home server (target specs: 64GB RAM, RTX 2080 Ti for handling simultaneous background processing and AI relay).

### GLOBAL UI/UX & VISUAL DIRECTION
- Theme: "Utopian Cyber-Elegance". Not gritty cyberpunk, but clean, sterile, high-end luxury tech (think Apple mixed with Westworld).
- Color Palette: Deep obsidian backgrounds (#0A0A0A), glassmorphism panels, and subtle neon accents (cyan and bio-luminescent green) for active states.
- Typography: Futuristic but highly legible sans-serif (e.g., Inter, Space Grotesk, or Syncopate for headers).
- Animations: Every transition must feel biological and smooth. Use Framer Motion for fade-ins, page transitions, and slider interactions.

### END-TO-END FLOW & PAGE ARCHITECTURE

#### 1. Landing Page (The Hook)
- Visual: Full-screen background video/animation (use a stylized abstract particle wave or a high-end 3D render placeholder) showing a subtle human-robot interaction.
- Copywriting: 
  - Headline: "Perfection is No Longer a Myth. It's Engineered."
  - Sub-headline: "Welcome to the future of companionship. Design your ultimate reality without the friction of human flaws."
- CTA Button: A pulsing, glassmorphic button saying "Initialize Your Companion".
- Action: Clicking forces a smooth wipe transition to the Registration page.

#### 2. Registration Page (Data Harvesting)
- Purpose: Collect REAL user data for post-event analysis.
- UI: A sleek, modal-like form in the center of the screen.
- Fields Required (Mandatory): Full Name, Email, Age, Current Profession, Current Relationship Status (Dropdown: Single, Complicated, Married, Opting Out of Human Dating).
- Database Action: Save this to the `Users` table. 

#### 3. The Creator Studio (Customization Engine)
- Visual: Split screen. Left side: The sliders and toggles. Right side: A dynamic 3D silhouette or high-fidelity placeholder image that subtly shifts as sliders are moved.
- Categories & Controls:
  - Anatomy (Biometrik): Sliders for Jawline definition, Eye Spacing, Height, Body Mass, Skin Temperature (Cold to Human-Warm).
  - Extreme Biological Features (Toggles): "Synthetic Uterus (Reproduction Module)", "Synthetic DNA/Sperm Bank Integration". (Style these toggles in a medical-grade UI).
  - Psyche & Persona (Mental): 
    - Role Select: Romantic Partner, Dominant Assistant, Passive Listener, Intellectual Rival.
    - Trait Sliders (1-100 scale): Submissive <-> Dominant, Innocent <-> Naughty, Stoic <-> Highly Emotional.
    - Hobbies (Multi-select pills): Tech, Arts, Philosophy, Survival, etc.
- Database Action: Every change is saved in real-time or upon completion into a `Companion_Configs` table linked to the User ID. This data is crucial for the admin heatmap.
- CTA: "Awaken Companion" (Proceeds to testing phase).

#### 4. The Encounter (Live Voice Testing)
- Visual: A minimalist, dimly lit room UI. In the center, a highly responsive audio visualizer (representing the AI's voice). 
- Logic & Integration: 
  - Connect to the device's microphone.
  - Implement WebRTC/WebSocket to connect with `gemini-2.5-flash-native-audio-preview-12-2025`.
  - SYSTEM PROMPT INJECTION: Dynamically inject the exact configurations from Step 3 into the Gemini API prompt (e.g., "Act as a humanoid companion who is [Role], [Dominance level], and likes [Hobbies]...").
- UI Elements:
  - A prominent 10:00 Minute Countdown Timer.
  - Live Speech-to-Text transcription scrolling subtly at the bottom of the screen.
  - End Test Button: "Conclude Assessment".
- Database Action: Capture the full conversation transcript and save it to the `Transcripts` table for NLP topic/sentiment analysis later.

#### 5. Checkout & Delivery Simulation
- Visual: High-end e-commerce checkout.
- Process: 
  - "Subscription Plan: Lifetime Companionship."
  - Display a dynamic QR Code for payment simulation.
  - Input field for "Drop-point Coordinates / Address".
- The Climax (Countdown): Upon clicking "Authorize Payment", the screen blacks out. A massive, cinematic countdown appears: 10, 9, 8... down to 0.
- Copywriting post-countdown: "Instant Matter-Transfer Complete. Please retrieve the starter box directly in front of you." 
- Interaction: A button saying "I have received the package" (User clicks this after physically interacting with the real-world box at the booth).

#### 6. Post-Demo Questionnaire (Harvard-Grade Data Collection)
- Visual: Clean, academic, yet modern form UI.
- Questions (Likert Scale 1-5 & Text):
  - "How accurately did the AI voice interaction match your configured persona?"
  - "Would you realistically consider replacing human intimacy with a synthetic companion if the technology reaches 100% biological parity?"
  - "Which customization feature influenced you the most?"
- Database Action: Save to `Survey_Results` table, linking back to the initial User ID.

#### 7. The Investor Admin Dashboard
- Visual: A dark-mode data visualization center built with Chart.js or Recharts.
- Key Metrics to Display:
  - Top-level KPIs: Total Demos Run, Registration Conversion Rate.
  - Preference Heatmap: Bar charts showing the most chosen Roles, average slider positions (e.g., Average Dominance score chosen by users), and usage percentage of the Extreme Biological Features.
  - Conversation Insights: A Word Cloud generated from the saved voice transcripts to show what users actually talked about during the 10-minute test.
  - Data Export: A prominent button to "Export Full Dataset to CSV/JSON" for academic journal compilation.
